Mujoco Native Installation

These install instructions assume you have already installed anaconda / miniconda

Windows

Run these commands line-by-line

conda create -y -n mujoco python=3.9
conda activate mujoco
conda install -y jupyterlab ffmpeg pip
pip install mujoco mediapy mujoco-python-viewer
pip install pygmsh
conda env config vars set MUJOCO_GL=glfw PYOPENGL_PLATFORM=glfw
conda deactivate && conda activate mujoco

Note: these installation instructions may not work depending on your available graphics card. Your other option is to use google colab.

Ubuntu

Run these commands line-by-line

conda create -y -n mujoco
conda activate mujoco
conda install -y python pip jupyterlab scipy ffmpeg
conda install -y -c conda-forge glew
conda install -y -c conda-forge mesalib
conda install -y -c anaconda mesa-libgl-cos6-x86_64
conda install -y -c menpo glfw3
conda install -y sympy pandas
pip install mujoco mediapy mujoco-python-viewer pygmsh
conda env config vars set MUJOCO_GL=osmesa PYOPENGL_PLATFORM=osmesa
conda deactivate && conda activate mujoco

MUJOCO_GL=egl and PYOPENGL_PLATFORM=egl is no longer working

Mac

conda create -y -n mujoco
conda activate mujoco
conda install -y python pip jupyterlab scipy ffmpeg pip
conda install -y sympy pandas
pip install mujoco mediapy mujoco-python-viewer pygmsh
conda install -y git pandoc #extra tools if you need them

Using mujoco

Before coding in your new mujoco environment, you will need to activate it.

conda activate mujoco
